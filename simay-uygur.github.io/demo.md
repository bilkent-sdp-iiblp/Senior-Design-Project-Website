## Pipeline Overview

Ubien starts by taking an uploaded source book. This source can be an IBL book or a regular book, and the parser extracts structure, content blocks, and key learning sections.

The content is then transformed into an inquiry-based course format. Sections, slides, and questions are organized into a guided learning flow so users can review and adjust the material before publishing.

During delivery, learners move through slides and questions with guided support. Animated avatars present slide content with Amazon Polly text-to-speech, while progress tracking records completion across course components.

If parsing results need correction, users can split and reorganize slides directly in the app. This keeps course preparation fast without forcing a full re-upload.
