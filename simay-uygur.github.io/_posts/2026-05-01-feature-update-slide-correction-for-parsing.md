date: 2026-05-01

Ubien now supports correction workflows for courses that were parsed incorrectly. If a course import creates slides with wrong boundaries or mixed content, users can fix the structure directly.

## Split and Correct Slides

When parsing results are not ideal, users can split slides and reorganize content into the correct slide sequence. This allows users to repair course flow without re-uploading the entire source file.

## Keep the Course Moving

The correction step makes course preparation more reliable by giving users control over final structure. Instead of being blocked by a parsing issue, users can quickly adjust slides and continue building the course.

## Why This Helps

This feature reduces friction in course creation and keeps both IBL and regular-book conversions practical for real classroom material.
