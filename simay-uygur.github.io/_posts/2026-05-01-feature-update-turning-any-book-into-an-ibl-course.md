Ubien now supports regular book uploads in addition to IBL books. This expands the earlier flow where the app focused on parsing books that were already written in an inquiry-based format.

## From Regular Book to IBL Material

When a regular book is uploaded, Ubien converts the content into IBL-style learning material. The generated material is organized around guiding questions and course steps so learners can work through the subject actively instead of only reading static text.

## Related Course Creation

After creating the IBL-style material, Ubien creates a related course inside the app. This course can include sections, slides, and questions that connect the uploaded book content to an inquiry-based learning flow.

## Why This Helps

This feature makes Ubien useful for a wider range of course materials. People can start from normal books, not only books that already follow IBL structure, and still create an inquiry-based online course.
